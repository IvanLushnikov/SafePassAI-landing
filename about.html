О проекте

SafePassAI — это pet-проект, направленный на повышение цифровой безопасности пользователей.

Функциональность:

  • Проверка логинов, email, имен и паролей в публичных утечках
  • Интеграция с API различных сервисов для получения информации об утечках
  • Агрегация и обработка результатов проверки
  • Генерация надёжных паролей при выявлении слабых или скомпрометированных
  • Асинхронное взаимодействие через Telegram-бота

Технологии:

  • Python
  • Работа с большими базами данных
  • Telegram Bot API
  • Интеграция с внешними API
  • Render и Vercel для хостинга

Цель:

Проект помогает улучшить навыки разработки, работы с API и повышает осведомлённость о цифровой безопасности.
