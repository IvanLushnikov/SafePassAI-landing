О проекте

SafePassAI — это pet-проект, направленный на повышение цифровой безопасности пользователей.

Функциональность:

  • Проверка логинов/email/имен/паролей в публичных словарях утечек
  • Подключение к API множества сервисов для получения информации об утечках
  • Обработка ошибок, агрегация результатов из разных источников
  • Генерация надёжных паролей при выявлении слабых или скомпрометированных
  • Асинхронная работа с пользователями через Telegram-бота

Технологии:

  • Python
  • Работа с большими локальными текстовыми базами
  • Telegram Bot API
  • Обработка и агрегация данных из внешнего API
  • Render и Vercel для хостинга и деплоя

Зачем:

Проект помогает отточить навыки работы с Telegram-ботами, API, обработкой данных и повысить осведомлённость пользователей о цифровой безопасности.
